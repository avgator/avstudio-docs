# Text Input Element

The `Text Input` element allows users to enter text data in your AVStudio interface.

## Overview

_Documentation coming soon._

This element provides a text input field for user interaction and data entry.

## How to Add a Text Input

1. Open your project in AVStudio and go to the `Editor` view.
2. From the elements panel, select `Text Input`.
3. The text input field will appear in your layout.
