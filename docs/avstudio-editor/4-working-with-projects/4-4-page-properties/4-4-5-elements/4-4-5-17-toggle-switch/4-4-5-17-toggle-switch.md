# Toggle Switch (Deprecated)

:::danger Deprecated
The Toggle Switch widget is deprecated and will be removed in a future release. It is retained for backward compatibility only and should not be used in new projects.
:::

## Status

- Maintenance: Critical fixes only
- New features: None planned
- Removal: Scheduled for a future version (date TBA)

## Existing Projects

You do not need to take immediate action; existing Toggle Switch instances will continue to function until removal. Plan to migrate during routine refactoring cycles.

## Migration Guidance (Optional)

1. Identify all pages containing Toggle Switch.
2. Record any dynamic bindings or actions attached to the widget.
3. Replace with current supported control(s) providing equivalent on/off functionality.
4. Reattach actions and verify layout and styling.
5. Test interaction and any dependent logic (signals, variables, parameters).

## Known Limitations

- Styling options are not being expanded.
- No further performance improvements will be applied.
- Not guaranteed to be compatible with future advanced theming features.
