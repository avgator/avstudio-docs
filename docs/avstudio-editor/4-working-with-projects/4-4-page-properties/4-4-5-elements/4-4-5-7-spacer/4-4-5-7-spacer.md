# Spacer Element

The `spacer` element provides spacing and layout control in your AVStudio interface.

## Overview

_Documentation coming soon._

This element helps with layout organization by providing empty space between other elements.

## How to Add a Spacer

1. Open your project in AVStudio and go to the `Editor` view.
2. From the elements panel, select `spacer`.
3. The spacer will appear in your layout.
