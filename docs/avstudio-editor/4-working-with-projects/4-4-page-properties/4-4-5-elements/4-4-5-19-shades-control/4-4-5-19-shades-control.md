# Shades Control Widget

The `Shades Control` widget provides shade and blind control functionality in your AVStudio interface. **(Premium Feature)**

## Overview

_Documentation coming soon._

This premium widget allows users to control window shades and blinds through the interface.

## How to Add Shades Control

1. Open your project in AVStudio and go to the `Editor` view.
2. From the elements panel under Widgets, select `Shades Control`.
3. The shades control widget will appear in your layout.

:::tip Premium Feature
This element requires a premium subscription to use.
:::
