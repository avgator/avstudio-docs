#  HTML5 Crestron Control System Configuration 

To enable HTML5 in the Crestron control system, execute (verify)
following commands:

```bash title="Terminal Commands" showLineNumbers
authentication on
userpageauth on
webserver allowsharedsession on
```

More details:
